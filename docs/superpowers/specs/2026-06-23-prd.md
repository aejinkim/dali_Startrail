# Dali Timetimer — Product Requirements Document

**Date:** 2026-06-23
**Status:** Approved

---

## Slogan

> "Your time is you."
> 시간이 곧 당신이다 — 당신이 시간을 어떻게 쓰는가가 곧 당신이 되어가는 과정이다.

---

## Problem Statement

기존 포모도로 앱은 타이머만 있다. 세션이 끝나면 데이터가 쌓이지만, 내가 오늘/이번 주 뭘 했는지 한눈에 보이지 않는다. 시각화가 부족해서 기록이 인사이트가 되지 못한다. 결국 앱을 꾸준히 쓸 이유가 없어진다.

Dali Timetimer는 시간 데이터를 쌓아서 "당신이 어떤 사람이 되어가고 있는지"를 보여주는 자기 실현 도구다.

---

## Target User

누구나 쓸 수 있는 범용 포모도로 앱 사용자.
자기 시간을 의미있게 쓰고 싶고, 그 기록이 자신을 이해하는 데이터가 되길 원하는 사람.

---

## Success Metrics

| 지표 | 목표 |
|------|------|
| Time to timer start | 앱 열고 타이머 시작까지 3초 이내 |
| Retention signal | 주간/월간 리포트를 기다리게 된다 |
| Shareability | 시각화가 아름다워 스크린샷 공유 욕구 발생 |

---

## Product Phases

### Phase 1 — Foundation (현재 스프린트)

기존 앱을 기반으로 핵심 UX를 완성한다.

| 기능 | 설명 | 스펙 문서 |
|------|------|----------|
| 25min 타이머 + 프리셋 | 슬라이더 max 25, 프리셋 버튼 추가 | `2026-06-23-feature-additions-design.md` |
| 브라우저 알림 | Notification API — 세션 종료 시 알림 | 동일 |
| 휴식 타이머 다이얼로그 | 세션 후 5분 휴식 제안 + Break 카운트다운 | 동일 |
| Analytics 시각화 업그레이드 | 기존 Analytics 탭 그래프/차트 개선 | 별도 스펙 작성 예정 |

### Phase 2 — Identity

데이터가 쌓이면서 나를 이해하는 도구로 발전한다.

| 기능 | 설명 |
|------|------|
| 주간 리포트 뷰 | "이번 주 나는 이런 사람이었다" — 활동별 시간, 목표 달성률, 패턴 |
| 목표(Goal) 시스템 | 장기 목표 설정 → 세션을 목표에 연결 → 목표별 진척도 시각화 |

### Phase 3 — Evolution (AI Coach)

타이머가 나를 알고, 내가 되고 싶은 사람이 되도록 돕는다.

| 기능 | 설명 |
|------|------|
| 패턴 학습 | 내 세션 데이터를 분석해 언제 집중이 잘 되는지, 어디에 시간을 쓰는지 파악 |
| 오늘의 제안 | 타이머를 열기 전에 "오늘 이걸 하세요"를 먼저 제시 |
| 스스로 진화하는 타이머 | 데이터가 쌓일수록 나에게 맞게 최적화되는 AI 코치 |

## Current App State

이미 구현된 기능:
- 원형 다이얼 타이머 (1~20분, 프리셋 5/10/20)
- 세션 목표 입력 + 메모
- 세션 기록 (localStorage + Supabase)
- Analytics 탭 (기본 집계)
- CSV 내보내기
- Auth (Supabase)

---

## Out of Scope (전 Phase 공통)

- 팀/공유 기능
- 모바일 앱 (native)
- 유료화 / 구독 모델
- 소셜 피드

---

## Design Principles

1. **Fast start** — 앱 열면 즉시 타이머. 입력은 선택.
2. **Beautiful data** — 시각화가 아름다워야 SNS 공유 욕구가 생긴다.
3. **Progressive depth** — 처음엔 단순한 타이머, 쓸수록 나를 이해하는 도구.
